The Engine

Engineering Context

This application demonstrates a production-grade Machine Learning Engineering lifecycle. It implements a decoupled N-Tier Architecture powered by a lightning-fast FastAPI backend. The architecture relies on Bio_ClinicalBERT for sequence classification (specialty prediction).

The deployment pipeline is fully automated and containerized using secure, multi-stage Docker builds hosted on Hugging Face Spaces. The entire experience is wrapped in a bespoke Vanilla JS & TailwindCSS frontend that highlights transparent Explainable AI (XAI) insights by dynamically annotating medical text with Captum Integrated Gradients.
