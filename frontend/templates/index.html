The Engine

Engineering Context

This application demonstrates a production-grade Machine Learning Engineering lifecycle. It implements a decoupled N-Tier Architecture powered by a lightning-fast FastAPI backend. The architecture relies on DualStreamFusionNER, which extracts spatial vision properties via EasyOCR and merges them with token embeddings from a fine-tuned Bio_ClinicalBERT Transformer via the Hugging Face Pipeline.

The deployment pipeline is fully automated and containerized using secure, multi-stage Docker builds hosted on Hugging Face Spaces. The entire experience is wrapped in a bespoke Vanilla JS & TailwindCSS frontend that highlights transparent Explainable AI (XAI) insights by dynamically annotating medical text.
