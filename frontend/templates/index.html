The Engine

Engineering Context

This application demonstrates a production-grade Machine Learning Engineering lifecycle. It implements a decoupled N-Tier Architecture powered by a lightning-fast FastAPI backend and ASGI server. To handle massive Transformer weights efficiently, the system completely decouples the source code from the model artifacts, dynamically streaming fine-tuned DistilBERT weights directly from a dedicated Hugging Face Model Registry at runtime.

The deployment pipeline is fully automated via GitHub Actions CI/CD (Pytest, Ruff) and containerized using secure, multi-stage Docker builds. The entire experience is wrapped in a bespoke Vanilla JS & TailwindCSS frontend that highlights transparent Explainable AI insights.
